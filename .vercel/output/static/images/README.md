# Images Directory

Place project images here with the following naming convention:
- `project-1.jpg`
- `project-2.jpg`
- `project-3.jpg`
- etc.

Recommended specifications:
- Format: JPG, PNG, or WebP
- Aspect ratio: 16:9
- Dimensions: 1200x675 pixels (or similar)
- File size: < 200KB (optimized for web)
